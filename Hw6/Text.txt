Язык программирования C++ нужен для разных целей, вот некоторые из них:
Разработка компьютерных игр и движков для игр. Благодаря скорости и 
производительности это один из лучших языков для создания игр. На C++ 
написаны Counter-Strike и StarCraft, также на нём работают движки Unity, 
Unreal Engine и ОС для PlayStation, Xbox. 
Разработка системного ПО. Этот язык является основой Adobe Photoshop, 
браузеров Mozilla Firefox и Internet Explorer, ОС Windows, программ из 
пакета Microsoft Office, движков поисковых систем Google и Yandex. 
GPU-вычисления. 
На C++ можно написать код для графического процессора, 
что позволяет распараллеливать сложные вычисления и 
добиваться высокой производительности и скорости. 
Разработка высоконагруженных и высокопроизводительных 
приложений. Язык незаменим для поисковых движков, 
роботов торговых бирж, систем управления базами 
данных и маршрутизации сетевого трафика.  
Кроме того, C++ может применяться математиками 
для решения сложных задач (работы с уравнениями, 
функциями, интер- и экстраполяциями). Физиками и 
химиками язык используется для моделирования сред 
и физических процессов. Музыкантами — для изучения 
акустических эффектов и даже написания музыки.
